# AI Task Prompts for Wedding Invitation System

This folder contains individual task prompts that can be executed by AI tools to build the Online Wedding Invitation System.

## Execution Order

Execute these tasks in the following order:

1. `01-supabase-setup.md` - Database and authentication setup
2. `02-database-schema.md` - Create database tables
3. `03-guest-landing-page.md` - Guest landing page with personalization
4. `04-invitation-page.md` - Main invitation page
5. `05-event-info-page.md` - Event information section
6. `06-rsvp-form.md` - RSVP form with validation
7. `07-album-gallery.md` - Photo album gallery
8. `08-wishes-section.md` - Guest wishes/messages
9. `09-qr-banking.md` - QR code banking section
10. `10-admin-auth.md` - Admin authentication
11. `11-admin-dashboard.md` - Admin dashboard
12. `12-admin-guests.md` - Guest management
13. `13-admin-rsvp.md` - RSVP management
14. `14-admin-wishes.md` - Wishes moderation
15. `15-admin-content.md` - Content management
16. `16-deployment.md` - Vercel deployment

## Tech Stack Reference

- **Frontend:** Next.js 16 (App Router) + TypeScript
- **Styling:** Tailwind CSS v4 + shadcn/ui
- **Animation:** Framer Motion
- **Backend:** Supabase (PostgreSQL)
- **Forms:** React Hook Form + Zod
- **Hosting:** Vercel
